In order to use these - you will need the bootimg-tools located here:
https://github.com/pbatard/bootimg-tools

and while it should be obvious, you will need the tools from your ADK in the path, we use both adb & fastboot and assume that you have set up your device for developer access, and that you have exported the files for slatekit nexus onto your device in the /data area (needs root access for this)

NB: No attempt has been made to make this work in an windows environment
